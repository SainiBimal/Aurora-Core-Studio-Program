# iPad GUI for Aurora RXC-3 (ReAX Core Studio)

A modern and responsive iPad graphical user interface (GUI) created using Aurora ReAX Core Studio for the Aurora RXC-3 control processor.

## Overview

This project provides a clean touchscreen interface designed for iPad devices to control AV systems powered by the Aurora RXC-3. The interface is optimized for ease of use, fast navigation, and responsive performance.

## Features

- 📱 iPad-optimized layout
- 🎨 Modern and clean UI design
- ⚡ Fast navigation between pages
- 🔊 Volume control
- 📺 Display control
- 💻 Source selection
- 🔌 Power On/Off controls
- 💡 Custom buttons and icons
- 🌐 Web-based interface published from ReAX Core Studio

## Software

- Aurora ReAX Core Studio
- Aurora RXC-3 Controller

## Requirements

- Aurora RXC-3 Controller
- ReAX Core Studio
- iPad with Safari or any modern web browser
- Network connection to the controller

## Usage

1. Open the project in Aurora ReAX Core Studio.
2. Modify the interface if needed.
3. Publish the project to the Aurora RXC-3 controller.
4. Open the controller's IP address from an iPad browser.
5. Add the page to the iPad Home Screen for a full-screen app experience.
